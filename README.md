-BananaPay
Sistema bancário web desenvolvido com Python/Flask e SQLite

-Funcionalidades

Cadastro de usuários com nome e CPF
Login com autenticação por sessão
Depósito com recarga automática de limite
Saque com uso de limite quando saldo insuficiente
Transferência entre contas com histórico para ambos os lados
Histórico de transações com data e destinatário

-Tecnologias

Python 3
Flask (Framework)
SQLite
HTML/CSS/JS

-Como rodar

  1.Clone o repositório
  2.Instale as dependências:
    -pip install flask

-Crie o banco de dados:

python init_db.py

-Inicie o servidor:

python run.py
aperte ctrl + clik no http://localhost:5000
